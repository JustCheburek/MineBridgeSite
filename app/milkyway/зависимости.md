# Зависимости переменных друг от друга

## Size ~ height

size = 3.5
height = 430 (1160 - 730)

size = 7
height = 490 (1250 - 760)

size = 10
height = 530 (1320 - 790)

Итого:
**height = (SIZE + 24.45) / 0.065**
